- npm i react-textarea-autosize

nanoid  (uid generator)
- npm i nanoid

zod (type protection)
- npm install zod

- npm install --save eventsource-parser

react communicate with firebase

firebase

zustand (global state for secure login status)

clsx (tailwind handling)

sweetalert2

3. Click 'Python: Create environment'
4. Click 'Venv'
5. Click the requirements.txt
6. Then open a new terminal, it will appears (.venv), indicate a virtual environment is built successfully
7. run "flask run"
